Simple AI Assistant

This simple AI Assistant is able to make jokes, tell the time, remind you about things, and search the web. Using speech recognition the AI will listen and wait for a command. As of now it only has a few commands and is very basic but I am using this a the first step on journey of learning Artificial Intelligence. I plan on building off of this project to make a more complex AI Assistant.

Assembly PONG

This is a clone of the game PONG using only Assembly Language. I wanted to use all of these projects to further my understanding of topics that interest me. By learning Assembly I am able to get a deeper understanding of the underlying computer architecture. I first planned on cloning Tetris but after breaking down the game I realized that would have to be the next Assembly Game I clone so I went with the slightly easier video game PONG.
